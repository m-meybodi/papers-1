69. A New Distributed Learning Automata Based Algorithm For Solving Stochastic Shortest Path Problem
Beigy, Hamid (Soft Computing Laboratory, Computer Engineering Departement, Amirkabir University of Technology); Meybodi, M.R. Source: Proceedings of the Joint Conference on Information Sciences, v 6, Proceedings of the 6th Joint Conference on Information Sciences, JCIS 2002, 2002, p 339-343
ISBN: 0970789017
Conference: Proceedings of the 6th Joint Conference on Information Sciences, JCIS 2002, Mar 8-13 2002, Research Triange Park, NC, United States Sponsor: Association for Intelligent Machinery;Information Sciences Journal;Duke University;Tamkang University
Publisher: Duke University/Association for Intelligent Machinery

Abstract: A new distributed learning automata (DLA) based algorithm for solving stochastic shortest path problem is presented. The objective is to use DLA to find a policy that determines a path from a source node to a destination node with minimal expected cost. It has been shown that the proposed algorithm finds the shortest path in a stochastic graph with probability as close as to unity. (11 refs.)
